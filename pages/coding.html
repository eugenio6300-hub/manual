코딩 팀 안내사항

우선 저는 이번 프로젝트에서 실무를 연습하기 위해 제가 직접 html 문서 작성에 참여하게 되었습니다. 코딩 팀의 업무는 다음과 같습니다.

  • 디자인 팀의 css에 맞춘 마크다운 문서 서식 개발
  • 마크다운 페이지 html 변환
  • 페이지 디버깅

  • 차후 유지보수 가능한 환경 구성

    사용 언어는 html. css입니다, java 사용도 고려하고 있기는 한데 지금으로서는 그렇게까지 필요할 것 같지 않으므로 고려만 해 두시길 바랍니다.
