텍스트 팀 공지사항

markdown이란?

텍스트를 html 문서로 변환하기 쉽게 하기 위해서는 파일의 확장자가 .md 인 마크다운 문서로 작업을 하셔야 합니다.

"전 머리털나서 마크다운이 뭔지 첨 들어보는데요"

그래서 지금부터 필요한 마크다운 문법만 가볍게 알려드리려고 합니다. 

# 제목
		챕터의 제목을 맨 위 헤더에 삽입할 때 사용

			## 부제목
			캐릭터 이름, 문단 제목 등을 표현하기 위해 사용

			### 제목 3
			캐릭터가 말할 때 이름 표기

			일반
			대사 표기

			**강조**
			강조되는 대사 표기. 인게임에서 강조된 대사 표시할때 사용

			|표|행|
			|---|---|
			|열|열|

		표를 표기하기 위함

챗지피티에서 본 것 같은데요:네 그렇습니다 그게 맞아요!!

이러한 마크다운 문서에 맞추어 캐릭터/도시사건 및 스토리/사건조사 등의 텍스트 md. 형식의 서식을 배포 예정이니. md문서 작동법을 어느정도 익혀두시길 바랍니다.

.md 파일의 경우. 컴퓨터 기본 메모장으로도 편집이 가능하니 별도의 앱을 설치할 필요는 없습니다.
